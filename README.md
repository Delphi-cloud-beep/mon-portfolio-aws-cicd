# 🚀 Portfolio Cloud & Pipeline CI/CD sur AWS

Bienvenue sur mon projet de portfolio d'ingénieure DevOps. Ce dépôt n'héberge pas seulement un site, il démontre une infrastructure automatisée.

## 🏗️ L'Architecture

- **Hébergement :** AWS S3 (Static Website Hosting).
- **Distribution :** AWS CloudFront (pour la performance et le HTTPS).
- **Backend Serverless :** API Gateway → AWS Lambda (Python) → DynamoDB.
- **Automatisation CI/CD :** GitHub Actions (déploiement continu à chaque push).

## 🛠️ Technologies utilisées

- **Cloud :** Amazon Web Services (AWS Certified Solutions Architect Associate).
- **DevOps :** GitHub Actions, Git.
- **Code :** Python (Scripts de monitoring), HTML/CSS (UI/UX).

## 📊 Monitoring

Le fichier `monitor.py` permet de vérifier la disponibilité du site en une ligne de commande.
